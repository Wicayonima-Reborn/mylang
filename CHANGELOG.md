## v0.1.0 — Safe Native Foundations

### Added

* Full compiler pipeline (lexer → parser → AST → semantic → borrow checker → codegen)
* Native x86_64 codegen (Windows Win64 & Linux SysV)
* Ownership & borrowing model
* Minimal runtime (string, print, clone)
* Basic language syntax (let, if/else, while, print)
* Examples folder with sample programs
* Cross-platform Makefile
* Documentation & README

### Known Limitations

* No user-defined functions yet
* No struct or trait system
* No IR or optimizations
* Limited diagnostics
